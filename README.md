# zsh-autoconfigure
My custom setup for oh-my-zsh.
## Included plugins
- git
- colored-man-pages
- jump
- sublime
- sudo
- themes
- copyfile
- copydir
- command-not-found
- zsh-syntax-highlighting
- zsh-autosuggestions
## Theme
`dst`
## Install
```bash
sh -c $(curl -fsSL https://raw.githubusercontent.com/ivni195/zsh-autoconfigure/main/install.sh)
```
