# zsh-autoconfigure
My custom setup for oh-my-zsh.
## Included plugins
- git
- colored-man-pages
- jump
- pip
- aliases
- sublime
- sudo
- docker
- copyfile
- copydir
- command-not-found
- zsh-syntax-highlighting
- zsh-autosuggestions
## Theme
`dst`
## Install
Run the command
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ivni195/zsh-autoconfigure/main/install.sh)"
```
When the shell initially changes, hit CTRL+D.
